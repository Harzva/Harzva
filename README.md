## About

我是一名专注于 **AI 系统与智能体基础设施** 的研究者与开发者。

我的旅程始于自动化工程（内蒙古工业大学，2019），在西安电子科技大学深耕计算机视觉研究，如今聚焦于 AI Agent 工具链的前沿领域。
